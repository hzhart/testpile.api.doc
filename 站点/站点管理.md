## 添加

```
POST /admin/station/v1/station
```

## 批量添加

```
POST /admin/station/v1/station/batch-create
```

> 参数: 使用上传excel接口解析上传的excel数据
> 参数: 同上添加接口, 数据格式为数组

```

```

## 删除

```

```
